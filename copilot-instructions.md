# GitHub Copilot Instructions

This root-level file is a discoverability pointer only.

GitHub Copilot loads repository instructions from `.github/copilot-instructions.md`, not from the repository root. The live Copilot adapter for this pack is therefore [`./.github/copilot-instructions.md`](./.github/copilot-instructions.md).

`CLAUDE.md` remains the authoritative rule source for all agents in this repository. If you copy this pack into another repository, copy `.github/copilot-instructions.md` as the actual instruction file and keep this pointer only if you want the adapter path to be visible from the root file list.
